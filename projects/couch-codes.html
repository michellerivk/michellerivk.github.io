Couch Codes

A couch party “Codenames-style” game: players use their phones (HTML) to choose actions while the host runs the Unity game.

Playable build (coming soon)

What I built

  • Unity host: game state, round flow, UI, and input handling.
  • NodeJS server: player sessions, real-time messaging, and synchronization.
  • Phone web client (HTML): lightweight UI for choosing actions and sending them to the host.

Technical highlights

  • Designed a clear state machine: lobby → in-round → resolution → next round.
  • Built a join flow (room code / session) and managed disconnect/reconnect behavior.
  • Kept the phone UI minimal to reduce friction for non-gamers.
